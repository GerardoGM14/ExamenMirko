Evaluación Unidad I

1. ¿Es posible usar CardViews directamente sobre un Activity o Fragment?

Respuesta correcta: Verdadero

2. ¿En qué generación se deja de utilizar la conmutación basada en circuitos para voz?

Respuesta correcta: 4G

3. En la evolución de las telecomunicaciones móviles se tienen en cuenta los siguientes aspectos:

Respuestas correctas: Impacto de calidad, Adopción de nuevas tecnologías, Desarrollo

4. ¿Qué elementos son importantes en cada proyecto de Android?

Respuestas correctas: bin/, res/, src/, sdk, build.xml

5. Es un archivo esencial en todas las aplicaciones. Se declara en el directorio raíz y contiene información sobre la aplicación que el sistema Android debe conocer antes de que se puedan ejecutar los códigos:

Respuesta correcta: AndroidManifest.xml

6. Android como sistema operativo para móviles fue construido sobre Linux Kernel (versión):

Respuesta correcta: 2.6

7. Es la capa que brinda interfaces estándares que exponen las capacidades de hardware del dispositivo al framework de la API de Java de nivel más alto:

Respuesta correcta: HAL

8. ¿En qué generación aparece la itinerancia internacional (roaming)?

Respuesta correcta: 1G

9. El panorama actual de las plataformas de desarrollo se caracteriza por:

Respuesta correcta: La coexistencia de propuestas nativas y multiplataforma, cada una de ellas con sus propias ventajas e inconvenientes.

10. ¿Cómo se llama la versión de Android que marca la pauta de los cambios de máquina virtual?

Respuesta correcta: LOLLIPOP
